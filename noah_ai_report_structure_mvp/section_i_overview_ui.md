# AI Report - Section I: System & Unit Overview (UI Blueprint)

This document outlines the conceptual UI structure for Section I of the AI-generated shift report.
It shows where data generated by `populate_section_i_system_overview()` would be displayed.

```html
<!-- Conceptual HTML-like structure -->
<div class="section-i-system-overview">
  <div class="header">
    <h2>Section I: System & Unit Overview</h2>
    <span class="current-datetime">{{ current_date_time }}</span> <!-- Populated by: current_date_time -->
  </div>

  <div class="shift-info-panel">
    <div class="info-item">
      <strong>Shift:</strong> <span id="shift-info">{{ shift_info }}</span> <!-- Populated by: shift_info -->
    </div>
    <div class="info-item">
      <strong>Reporting Nurse:</strong> <span id="reporting-nurse">{{ reporting_nurse }}</span> <!-- Populated by: reporting_nurse -->
    </div>
    <div class="info-item">
      <strong>Receiving Nurses:</strong>
      <ul id="receiving-nurses-list">
        <!-- Loop through 'receiving_nurses' array -->
        <li>{{ nurse_name_1 }}</li> <!-- Populated by: receiving_nurses[0] -->
        <li>{{ nurse_name_2 }}</li> <!-- Populated by: receiving_nurses[1] -->
        <!-- ... etc. -->
      </ul>
    </div>
  </div>

  <div class="unit-status-panel">
    <h3>Unit Status</h3>
    <div class="info-item">
      <strong>Census Overview:</strong>
      <p id="unit-census-overview">{{ unit_census_overview }}</p> <!-- Populated by: unit_census_overview -->
    </div>

    <div class="info-item alerts">
      <strong>Urgent Unit Alerts/Notes:</strong>
      <ul id="urgent-unit-alerts-list">
        <!-- Loop through 'urgent_unit_alerts' array -->
        <li>{{ alert_message_1 }}</li> <!-- Populated by: urgent_unit_alerts[0] -->
        <li>{{ alert_message_2 }}</li> <!-- Populated by: urgent_unit_alerts[1] -->
        <!-- ... etc. -->
      </ul>
      <!-- Could also be a free-text input for nurses to add unit-level notes -->
      <textarea placeholder="Add new unit-level alert or note..."></textarea>
      <button>Add Unit Note</button>
    </div>
  </div>
</div>

<!--
Data Mapping:
- current_date_time -> {{ current_date_time }}
- shift_info -> {{ shift_info }}
- reporting_nurse -> {{ reporting_nurse }}
- receiving_nurses -> Loop to create list items like {{ nurse_name_X }}
- unit_census_overview -> {{ unit_census_overview }}
- urgent_unit_alerts -> Loop to create list items like {{ alert_message_X }}
-->
```
